TeamFlow – Full Stack Task Management System

TeamFlow is a full-stack Kanban-style project management application built using:

Frontend:
- React
- TailwindCSS
- Drag & Drop Kanban Board

Backend:
- FastAPI
- SQLAlchemy
- JWT Authentication

Features:
- User login authentication
- Create and delete projects
- Create, edit, and delete tasks
- Drag and drop Kanban board (To Do / In Progress / Completed)
- REST API with FastAPI
- Swagger API testing

Tech Stack:
Frontend: React + Vite + Tailwind
Backend: FastAPI + SQLAlchemy
Database: SQLite
