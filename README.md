TeamFlow – Full Stack Task Management System

TeamFlow is a full-stack Kanban-style project management application built using:

Frontend:
- React
- TailwindCSS
- Drag & Drop Kanban Board

Backend:
- FastAPI
- SQLAlchemy
- JWT Authentication

Features:
- User login authentication
- Create projects
- Create and delete tasks
- Drag and drop Kanban board (To Do / In Progress / Completed)
- REST API with FastAPI
- Swagger API testing

Tech Stack
- React
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT Authentication
- Render Deployment
- Vercel Deployment

Frontend: https://teamflow-six-sigma.vercel.app
